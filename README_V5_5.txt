Machawi Dajaj ERP Cloud V5.5 — نسخة الاستقرار الأولى

ما تم إصلاحه:
1. إصلاح كود CSS غير صالح في بداية index.html.
2. إضافة manifest.json الصحيح.
3. إضافة pwa-register.js.
4. إضافة Service Worker للعمل كتطبيق PWA ودعم العمل دون اتصال بعد أول فتح.
5. إضافة بيانات التطبيق وألوان الهوية في رأس الصفحة.
6. تحديث رقم النسخة إلى V5.5.

طريقة التجربة:
- للتجربة السريعة افتح index.html.
- لتثبيت التطبيق كـ PWA يجب رفع المجلد إلى Vercel أو تشغيله عبر خادم محلي؛ التثبيت لا يعمل عند فتح الملف مباشرة بصيغة file://.

مهم:
- البيانات الحالية ما زالت محفوظة في المتصفح LocalStorage.
- لا تحذف بيانات المتصفح قبل أخذ نسخة احتياطية من داخل البرنامج إن كانت وظيفة التصدير متاحة.
