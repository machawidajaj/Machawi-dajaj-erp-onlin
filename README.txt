Machawi Dajaj ERP Online — Supabase

1. افتح Supabase > SQL Editor.
2. افتح ملف supabase_setup.sql وانسخ محتواه كاملاً.
3. اضغط Run.
4. ارفع index.html الجديد إلى GitHub بدلاً من الملف القديم.
5. افتح الموقع على الجهاز الذي يحتوي على البيانات الحالية أولاً.
6. انتظر حتى تظهر عبارة «تم الحفظ» أو «متصل بالسحابة».
7. افتح نفس الموقع على الهاتف أو الحاسوب الآخر.

مهم:
- يحتفظ البرنامج بنسخة محلية احتياطية.
- المزامنة السحابية تتم تلقائياً كل عدة ثوانٍ.
- هذه مرحلة أولى تعتمد على سجل JSON مركزي.
- الحماية المتقدمة بحسابات Supabase Auth ستكون المرحلة التالية.
