Machawi Dajaj ERP PWA

ارفع هذه الملفات كلها إلى نفس مستودع GitHub:
- index.html
- manifest.json
- service-worker.js
- مجلد icons

بعد رفعها:
1. انتظر Vercel حتى ينتهي النشر.
2. افتح الموقع في Chrome.
3. من قائمة Chrome اختر تثبيت التطبيق.
4. أو من داخل الإعدادات اضغط زر تثبيت التطبيق عندما يظهر.

مهم:
- التطبيق يبقى متصلاً بـ Supabase.
- Service Worker لا يخزن طلبات Supabase.
- صفحة التطبيق الأساسية يمكن فتحها حتى عند انقطاع الإنترنت، لكن البيانات السحابية تحتاج اتصالاً.
