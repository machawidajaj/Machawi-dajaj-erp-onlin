التعديلات:
- حذف أزرار إضافة وتعديل التحويل والمنتجات من صفحة التشغيل.
- إنشاء صفحة مستقلة داخل الإدارة: إعدادات التحويل والإنتاج.
- الصفحة الجديدة مخصصة لصلاحية شخص آخر.
- إضافة وتعديل وحذف أنواع التحويل تعمل داخل الصفحة الجديدة.
- إضافة وحذف منتجات التحويل تعمل داخل الصفحة الجديدة.
- فلترة تقرير التحويل حسب اليوم، أمس، الأسبوع، الشهر، أو فترة مخصصة.
- الفلتر الافتراضي يعرض دفعات اليوم.
- حذف عمود التاريخ المكرر؛ وقت بداية العملية يتضمن التاريخ والساعة.
